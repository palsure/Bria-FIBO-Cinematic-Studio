# Features Documentation

## Core Features

### 2-Step Workflow

#### Step 1: Scripting
- **Text Input**: Paste your script directly into the text area
- **File Upload**: Upload a script file (.txt format)
- **Auto-Generation**: Storyboard automatically generates after script parsing
- **Scene Extraction**: Automatically identifies and extracts scenes from script

#### Step 2: Scene Generation and Editing
- **Scene Review**: View all generated scenes with descriptions
- **Parameter Customization**: Adjust camera, lighting, color, and composition
- **Individual Scene Editing**: Edit and regenerate individual scenes
- **HDR Support**: Enable 16-bit HDR output for professional workflows
- **Export Options**: PDF and Animatic export

### Scene Management

#### Save Individual Scenes
- Save any scene to your library
- Access saved scenes from "My Scenes" tab
- View scene details, parameters, and descriptions
- Delete scenes when no longer needed

#### Save Storyboards
- Save complete storyboards with a custom name
- Edit storyboard name after saving
- Load saved storyboards for editing
- Delete storyboards from library

### Parameter Control

#### Camera Parameters
- **Angle**: eye_level, high, low, bird_eye, etc.
- **FOV**: Field of view (degrees)
- **Movement**: push_in, pull_out, pan, tilt, etc.

#### Lighting Parameters
- **Time of Day**: day, night, golden_hour, blue_hour
- **Style**: natural, dramatic, soft, harsh
- **Direction**: front, side, back, top

#### Color Parameters
- **Palette**: neutral, warm, cool, desaturated, vibrant
- **Saturation**: 0.0 to 1.0

#### Composition Parameters
- **Framing**: close_up, medium, wide, extreme_wide
- **Rule of Thirds**: Enable/disable
- **Depth of Field**: shallow, medium, deep

### Export Features

#### PDF Export
- Professional storyboard layout
- Scene descriptions included
- High-quality image rendering

#### Animatic Export
- Video sequence from storyboard
- Configurable duration per scene
- Professional video format

## User Interface Features

### Tab Navigation
- **Create Storyboard**: Main creation workflow
- **My Storyboards**: View and manage saved storyboards
- **My Scenes**: View and manage saved individual scenes

### Notifications
- Success notifications (green)
- Error notifications (red)
- Warning notifications (orange)
- Info notifications (blue)
- Auto-dismiss after 3 seconds
- Manual close option

### Responsive Design
- Works on desktop and mobile
- Adaptive layouts
- Touch-friendly controls
